## 📌 Project Overview

This project is an end-to-end Machine Learning + NLP system that analyzes Flipkart product reviews and classifies them into Positive or Negative sentiment.

It also includes a deployed Streamlit web application where users can input reviews and get instant predictions.

---

## 🎯 Key Features

- Text Cleaning & Preprocessing  
- Feature Engineering using TF-IDF  
- Handling class imbalance  
- N-grams for better context understanding  
- Logistic Regression model  
- Streamlit Web App  
- Real-time prediction with confidence score  

---

## 📊 Dataset

- Source: Flipkart Reviews  
- Size: 20,000+ reviews  
- Features:
  - Review Text  
  - Ratings  
  - Product Category  

---

## 🧠 Model Performance

- Accuracy: ~92%  
- F1 Score: ~0.91  

Challenge: Imbalanced dataset  
Solution: Used class_weight='balanced' and n-grams
